# Project README

## Project Overview

A demo app built with the react native sdk.

---

## Installation & Setup

```bash
git clone https://github.com/Jamal-09/hng_auth_demo
cd hng_auth_demo
```

```bash
npm install
npx expo prebuild --clean
```

```bash
npm run:android
npm run:ios
```
