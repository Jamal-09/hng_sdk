# Project README

## Project Overview

A demo app built with the react native sdk.

---

## Installation & Setup

```bash
git clone https://github.com/Jamal-09/hng_sdk
cd ja-sdk-demo
```

```bash
npm install
npx expo prebuild
```

```bash
npx expo run:android
npx expo run:ios
```
